# T-News - Социальная платформа для общения

T-News - это современная социальная платформа, созданная для публикации новостей, обмена мыслями и взаимодействия между пользователями. Приложение предоставляет функционал публикации постов, комментирования, лайков и управления профилем.

## Основные возможности

### Функционал пользователя

- **Регистрация и авторизация** через Firebase Authentication
- **Создание и публикация постов** с текстовым содержимым
- **Система лайков** - возможность оценивать посты других пользователей
- **Комментирование** - обсуждение постов в реальном времени
- **Управление профилем** - настройка имени, био и аватара
- **Поиск постов** по содержанию и авторам

### Технические особенности

- Real-time обновления через Firebase Firestore
- Адаптивный дизайн для всех устройств
- Оптимизированная производительность
- Безопасная аутентификация и авторизация
- Клиентская валидация форм

## Технологический стек

### Frontend

- **React 18** - пользовательский интерфейс
- **TypeScript** - типизация и надежность кода
- **React Router DOM** - навигация между страницами
- **CSS Modules** - стилизация компонентов

### Backend & Infrastructure

- **Firebase Authentication** - управление пользователями
- **Cloud Firestore** - база данных реального времени
- **Firebase Storage** - хранение файлов
- **Vercel** - хостинг и деплой
